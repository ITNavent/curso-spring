# Curso Spring Framework
Este proyecto tiene como finalidad dar una introducción al uso de Spring Framework. Muestra el uso de las principales características de Spring mediante diversos ejemplos.

## spring-basics
El proyecto muestra las funcionalidades básicas de spring core:
* Inyección de dependencias
* Componentes de Spring
* Creación de Beans
* Configuración
* Externalización
* Mensajes
* Spring Boot
* Test de integración

## spring-command-line
Muestra una aplicación de línea de comandos y algunos usos avanzados de inyección de dependencias.

## spring-rest
Muestra una aplicación Api REST construida con Spring MVC + Spring Data con MongoDB
* Controllers
* DataBinding + Validaciones
* Cache
* Rest Template
* Repository
* Swagger API
* Manejo de Exceptions
